# WatchParty 🎬

A real-time watch party app — watch YouTube videos together in sync.

## Structure

| Folder | Description |
|--------|-------------|
| `/frontend` | React + Vite + Tailwind CSS |
| `/backend` | Node.js + Express + Socket.IO + PostgreSQL |

## Frontend (Live)
🔗 [Click here to view the live app](https://ytwatchparty.vercel.app/)

## Tech Stack

**Frontend:** React 18, Vite, Tailwind CSS, Socket.IO Client  
**Backend:** Node.js, Express, Socket.IO, PostgreSQL, MongoDB, JWT

## Run Locally

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
# add your .env file
npm run dev
```
